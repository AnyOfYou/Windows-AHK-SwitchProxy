#Windows-AHK-SwitchProxy

Windows下快速切换IE(系统)代理的工具，被[小众软件](http://www.appinn.com)所[发布](http://www.appinn.com/ahk-dary-ie-pr/)

* 直接运行为切换代理状态。也可传入(address,state)，来指定具体的代理状态，如(127.0.0.1:9999,ON)
